Un minuteur Pomodoro sur votre bureau, flashé en un clic.

Pomodoro CYD transforme un écran ESP32 « Cheap Yellow Display » à ~8 € en minuteur de concentration tactile : durées réglables du bout du doigt, horloge NTP, configuration Wi-Fi depuis le navigateur et portail web local. Aucun IDE, aucun câblage — juste un câble USB.

Web flasher

Installez le firmware depuis votre navigateur

Le flash utilise Web Serial (ESP Web Tools) : pas de drivers exotiques, pas de ligne de commande. Une fois flashé, l'assistant vous propose directement de connecter le Pomodoro à votre Wi-Fi.

Avant de cliquer

  1. Utilisez Chrome ou Edge (Firefox et Safari ne gèrent pas Web Serial).
  2. Branchez le CYD en USB avec un câble data (pas un câble de charge seule).
  3. Cliquez, choisissez le port série, puis Install.
  4. À la fin, renseignez votre Wi-Fi dans la fenêtre — le lien vers le portail de configuration s'affiche.
Navigateur non compatible — ouvrez cette page dans Chrome ou Edge. Le flash nécessite une page servie en HTTPS.

Matériel

Un seul composant

Le projet tourne sur le fameux « CYD » : un ESP32 avec écran tactile résistif 2.8 pouces intégré, trouvable pour moins de 10 €.

requis

ESP32-2432S028R — « Cheap Yellow Display »

  • Écran 2.8" ILI9341 240×320, tactile XPT2046
  • ESP32 Wi-Fi + Bluetooth, LED RGB embarquée
  • Alimentation par simple câble micro-USB
  • Variantes v2 (double USB) et v3 supportées
optionnel

Boîtier imprimé en 3D

De nombreux boîtiers pour CYD existent sur Printables et MakerWorld (recherchez « ESP32-2432S028R case »). Un support incliné à 30–45° est idéal pour un minuteur de bureau.

Fonctionnalités

Minuteur tactile complet

  • Cycle focus / pause courte / pause longue
  • Durées réglables à l'écran (boutons + / –)
  • Points de progression du cycle, compteur de sessions
  • Démarrer, pause, reset, passer — au doigt

Connecté, sans cloud

  • Configuration Wi-Fi depuis le navigateur (Improv Serial)
  • Horloge locale synchronisée par NTP
  • Accessible via pomodoro.local
  • Tout reste sur votre réseau — zéro service externe

Signal de fin de phase

  • Flash de la LED RGB embarquée en fin de session
  • Couleur de l'anneau selon la phase en cours
  • Enchaînement automatique des phases (optionnel)

Pour les bidouilleurs

  • Endpoint /api/state en JSON — parfait pour Home Assistant
  • Réglages persistants en mémoire NVS
  • Code source PlatformIO + LVGL 9, licence libre

Portail web

Configurez tout depuis le navigateur

Une fois connecté au Wi-Fi, le Pomodoro expose une page locale de configuration — sans reflasher, sans application à installer.

Minuteur

Durées focus, pause courte, pause longue, nombre de sessions par cycle et enchaînement automatique. Les changements s'appliquent immédiatement à l'écran.

Écran & LED

Luminosité du rétroéclairage, activation du flash LED de fin de phase et fuseau horaire pour l'horloge NTP.

Wi-Fi

Changez de réseau à tout moment depuis le portail, ou reprovisionnez par USB via le flasher de cette page.

API d'état

GET /api/state renvoie phase, état, temps restant et sessions terminées — intégrable en capteur RESTful dans Home Assistant.

Dépannage

Questions fréquentes

Le bouton de flash ne détecte aucun port série
Vérifiez que vous utilisez Chrome ou Edge, et surtout que votre câble USB transporte les données (beaucoup de câbles fournis avec des gadgets sont « charge seule »). Sous Linux, ajoutez votre utilisateur au groupe dialout. Le CYD utilise une puce CH340 : sous Windows, le pilote s'installe normalement tout seul.
Le flash échoue à « Connecting… »
Maintenez le bouton BOOT du CYD enfoncé au moment où le flash démarre, puis relâchez. Fermez aussi tout moniteur série ouvert (Arduino IDE, PlatformIO) qui occuperait le port.
La connexion Wi-Fi échoue après avoir saisi le mot de passe
L'ESP32 ne supporte que le Wi-Fi 2,4 GHz. Vérifiez que votre réseau n'est pas en 5 GHz uniquement, que le mot de passe est correct et que le signal est suffisant. Les portails captifs et l'isolation client des réseaux invités posent aussi problème.
pomodoro.local ne répond pas
Le mDNS est parfois bloqué par les routeurs ou l'isolation des VLAN. Utilisez directement l'adresse IP affichée en haut à gauche de l'écran du CYD.
L'écran reste noir ou les couleurs sont inversées
Il existe plusieurs révisions du CYD. Si vous avez le modèle à double port USB (micro-USB + USB-C), flashez la variante v2 du firmware (disponible dans les releases GitHub du projet), qui corrige l'inversion des couleurs et le rétroéclairage.
L'heure affichée est incorrecte
L'horloge se synchronise par NTP à la connexion Wi-Fi. Le fuseau par défaut est Europe/Bruxelles ; il se change dans le portail web (champ « Fuseau horaire », format POSIX TZ).
